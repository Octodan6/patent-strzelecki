# Patent strzelecki PZSS — materiały do nauki

Zestaw do samodzielnej nauki do części teoretycznej egzaminu na patent strzelecki PZSS.

## Co jest w środku

| Plik | Co to jest |
|---|---|
| `index.html` | **Interaktywny quiz** na telefon: pytania ABC jedno po drugim, zielone/czerwone podświetlenie, egzamin próbny (10 pytań jak na prawdziwym egzaminie) i podsumowanie błędów po każdej sesji |
| `streszczenie.md` | Kompendium całego materiału (czytelne bezpośrednio na GitHubie) |

| `plan-nauki.md` | Plan nauki na 14 dni |

## Jak umieścić to na GitHubie (krok po kroku)

1. Wejdź na **github.com** i zaloguj się (lub kliknij **Sign up (Zarejestruj się)** i załóż darmowe konto).
2. Kliknij zielony przycisk **New (Nowy)** przy liście repozytoriów — albo znak **+** w prawym górnym rogu i wybierz **New repository (Nowe repozytorium)**.
3. W polu **Repository name (Nazwa repozytorium)** wpisz np. `patent-strzelecki`.
4. Zaznacz **Public (Publiczne)** — to konieczne, żeby darmowa strona z quizem działała. Zaznacz też **Add a README file (Dodaj plik README)**.
5. Kliknij **Create repository (Utwórz repozytorium)**.
6. W repozytorium kliknij **Add file (Dodaj plik)** → **Upload files (Prześlij pliki)**.
7. Przeciągnij wszystkie pliki z tego zestawu (`index.html`, `streszczenie.md`, `streszczenie.pdf`, `plan-nauki.md`, `README.md`) na pole przesyłania.
8. Na dole kliknij zielony przycisk **Commit changes (Zatwierdź zmiany)**.

## Jak włączyć quiz jako stronę internetową (GitHub Pages)

1. W repozytorium kliknij zakładkę **Settings (Ustawienia)** — na górze, po prawej.
2. W menu po lewej stronie kliknij **Pages (Strony)**.
3. W sekcji **Build and deployment (Budowanie i wdrażanie)**, w polu **Source (Źródło)** wybierz **Deploy from a branch (Wdróż z gałęzi)**.
4. W polu **Branch (Gałąź)** wybierz **main**, obok zostaw **/ (root)** i kliknij **Save (Zapisz)**.
5. Odczekaj 1–2 minuty i odśwież stronę. U góry pojawi się adres w stylu:
   `https://twoja-nazwa.github.io/patent-strzelecki/`
6. Otwórz ten adres **na telefonie** — uruchomi się quiz. Dodaj go do ekranu głównego:
   - iPhone: przycisk **Share (Udostępnij)** → **Add to Home Screen (Dodaj do ekranu początkowego)**,
   - Android (Chrome): menu **⋮** → **Add to Home screen (Dodaj do ekranu głównego)**.

Od tej pory masz quiz pod ręką jak zwykłą aplikację — działa w każdej wolnej chwili.

Materiał opracowano na podstawie kursu patentstrzelecki.eu, Ustawy o broni i amunicji, Kodeksu karnego oraz regulaminów PZSS/ISSF (stan: 2026). Regulamin patentowy PZSS zmieniał się w ostatnich latach — przed egzaminem sprawdź aktualną wersję na pzss.org.pl.
